# svt-val

This repository contains a Selenium-based workflow that steals election polling data from https://www.svt.se/datajournalistik/bygg-en-valjare/

> [Download latest polling data (CSV)](https://github.com/thor-computer/svt-val/releases/latest/download/csv.csv)
